## Budget Tracker
Budget Tracker is the program that record all your spending.
The inspiration about this project is I dont know to to save my money. So, I write this code

# Installation
```sh
 # Download Requirements
 pip install -r requirements.txt

 # To run the program, you should create Virtual Environement for Python
 # After, you have running VENV, running
 python3 budget_tracker.py
```
